# Gromov-Witten-Invariant
To calculate Gromov-Witten Invariant

```
├── src
│   ├── compressed_class_list
│   │    ├── compressed_class_list.ipynb - contains code for defining compressed class list
│   │    ├── test_compressed_class_list.ipynb - contains code for testing compressed class list
│   ├── gw
│   │    ├── gw_exp.ipynb - contains code for computing gromov-witten invariant of complex projective space
│   │    ├── cp2_recursion.ipynb - contains code for computing recursion for CP2
│   │    ├── test_gw.ipynb - contains code for testing gromov-witten invariant of complex projective space
│   ├── ogw
│   │    ├── ogw_exp.ipynb - contains code for computing open gromov-witten invariant of projective space
│   │    ├── test_ogw.ipynb - contains code for testing open gromov-witten invariant of projective space
│   ├── chiang
│   │    ├── chiang_ogw.ipynb - contains code for testing open gromov-witten invariant of Chiang Lagrangian
│   │    ├── chiang_experiment.ipynb - contains code for computing various values of open gromov-witten invariant of Chiang Lagrangian
                                       and some related experiment
```
